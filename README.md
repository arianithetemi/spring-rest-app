# Spring RESTful Application
A RESTful API Application built on Spring Framework. The resource of REST API is **People**, which will response JSON structure data based on endpoints.

## Technologies used:
  * Language: **Java**
  * Development Kit: **Java SDK 8**
  * Framework: **Spring Framework**
  * Build Automation Tool: **Apache Maven**
  * Database: **H2 (In-Memory)**
  * ORM: **Hibernate**
  * Unit Testing Framework: **JUnit**
  * Service API: **REST**
  * REST Testing Library: **REST Assured**
